## Что сделано

- На `/catalog/:category` добавлены хлебные крошки, `h1` категории и переключатель режимов «Все товары / Бренды».
- Режим брендов включается через `?view=brands`; для экрана стоматологических установок доступен URL `/catalog/stomatologicheskie-ustanovki?view=brands`.
- Добавлен `BrandSearch` — правая область с поиском по брендам, счётчиком найденных брендов и 4-колоночной сеткой карточек на desktop.
- Добавлен `catalogBrands` — статический слой брендовых карточек по категориям с fallback-добавлением брендов из товаров.
- Карточки брендов кликабельны, имеют wordmark-логотип, подпись категории, hover-подъём, тонкую обводку и мягкую тень.
- Боковая навигация категорий сохранена слева и закрепляется на desktop.

## Проверка

- `npm run build` — успешно; осталось только существующее предупреждение Vite о крупном chunk.
- Desktop 1440px: активен режим «Бренды», 8 брендов, первая строка сетки — 4 карточки, горизонтального скролла нет.
